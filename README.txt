Easy switch to any site that you manage.

== Description ==

This plugin create a menu with the links of all WordPress sites that you have (and that uses this plugin, obviously).

Easy jump from any site that you have been signed-up.

== Installation ==

1. Upload `mysites.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. All done!

Do the same in any site wath you want.

A menu will be displayed under profile link, at top right of screen.
